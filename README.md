📚 无人驾驶汽车用户调研分析 | MySQL 数据项目
项目简介：本项目基于 137 份无人驾驶汽车用户调研问卷数据，使用 MySQL 完成从数据清洗、用户画像、购买意愿分析、顾虑因素挖掘到交叉维度对比的全流程分析。目标：定位用户核心顾虑、识别高意向客群、量化体验对决策的影响，为市场推广、产品优化与用户教育提供数据支撑。
🎯 分析目标

    清洗数据，统一字段、修正错别字、保证分析质量
    构建调研用户基础画像（性别、年龄、学历、收入）
    分析整体购买意愿，量化市场接受度
    搭建转化 / 接受漏斗，定位最大阻碍环节
    多维度对比转化率 / 接受度差异（年龄、性别、学历、体验与否）
    挖掘高意愿用户与观望 / 拒绝用户关键特征
    输出可落地的运营、产品、营销优化建议

📊 数据集说明

    来源：无人驾驶汽车用户调研问卷（脱敏、可本地安全使用）
    表名：autonomous_car_survey
    关键字段：用户基本信息、了解程度、购买意愿、体验经历、满意度、购车理由、拒绝顾虑、改进建议等

🛠 使用工具

    Excel / Tableau（可视化）
    分析思路：用户画像 + 意愿漏斗 + 交叉归因 + 分群洞察

🔎 核心结论

    市场接受度极高：整体购买意愿超 80%，市场潜力巨大
    核心客群清晰：18–35 岁、高学历群体接受度最高，是首要目标人群
    体验是关键转化抓手：体验过无人驾驶的用户，购买意愿显著更高
    最大痛点是安全与技术：安全顾虑、技术不成熟是最主要拒绝原因
    认知不足是普遍瓶颈：用户整体了解程度偏低，科普与体验是破局关键
